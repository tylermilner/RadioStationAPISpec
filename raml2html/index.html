    /config

    Station configuration.

    GET /config

    Fetch the current station configuration.

    GET /config

    Response 200

    Body

    Media type: application/json

    Type: object

    Properties:

    • id: required (string)
    • streams: required (array of )
    • stationWebsiteUrl: required (string)

    Example:

    {
  "id": "1",
  "streams": [
    {
      "name": "AAC+ high quality US",
      "extension": "aac+",

    },
    {
      "name": "MP3 standard quality US",
      "extension": "mp3",

      "qualityScore": 50
    }
  ],
  "stationWebsiteUrl": "http://bassdrive.com/"
}

    /login

    Authenticate with the station. Used by DJs or admins that have been manually registered with the station.

    POST /login

    Login to the station using username/password credentials to obtain a session token.

    POST /login
    Body

    Media type: application/x-www-form-urlencoded

    Type: object

    Properties:

    • email: required (string)
    • password: required (string)

    Example:

    {
  "email": "dj1@stationdomain.com",
  "password": "abc123"
}

    Response 200

    Body

    Media type: application/json

    Type: object

    Properties:

    • token: required (string)

    Example:

    {
  "token": "eyJhbGciOiJIUzI1NiIsInR5cCI6IkpXVCJ9.eyJpc3MiOiJzY290Y2guaW8iLCJleHAiOjEzMDA4MTkzODAsIm5hbWUiOiJDaHJpcyBTZXZpbGxlamEiLCJhZG1pbiI6dHJ1ZX0.03f329983b86f7d9a9f5fef85305880101d5e302afafa20154d094b229f75773"
}

    Response 401

    401 Unauthorized response indicates that the credentials were not valid.

    /nowPlaying

    Now playing information.

    GET /nowPlaying

    Fetch the current now playing information.

    GET /nowPlaying

    Response 200

    Body

    Media type: application/json

    Type: object

    Properties:

    • id: required (string)
    • djName: required (string)
    • showName: required (string)
    • trackTitle: (string)
    • trackArtist: (string)

    Example:

    {
  "id": "1",
  "djName": "Stunna",
  "showName": "The Greenroom",
  "trackTitle": "Can't Stop",
  "trackArtist": "Command Strange"
}

    Headers

    Body

    Media type: application/json

    Type: object

    Properties:

    • id: required (string)
    • djName: required (string)
    • showName: required (string)
    • trackTitle: (string)
    • trackArtist: (string)

    Example:

    {
  "id": "1",
  "djName": "Stunna",
  "showName": "The Greenroom",
  "trackTitle": "Can't Stop",

    /shows

    Retrieve/edit all of the station's shows.

    GET /shows

    Get a list of Shows.

    GET /shows
    Query Parameters
    • dayOfWeek: required (one of mon, tues, wed, thurs, fri, sat, sun)
    • isActive: required (boolean)

    Response 200

    Body

    Media type: application/json

    Type: array of Show

    Items: Show

    • description: required (string)
    • broadcastInfo: required (object)
      • name: required (string)
      • djId: required (string)
      • isActive: required (boolean)
      • id: required (string)
      • nextBroadcastTime: required (integer)
      • soundcloudUrl: required (string)
      • avatarUrl: required (string)
